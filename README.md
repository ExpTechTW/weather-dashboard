# 複合式防災電子看板

## 參數

### region

- 說明: 欲設定區域之郵遞區號。[點擊查詢](https://raw.githubusercontent.com/ExpTechTW/API/refs/heads/master/resource/region.json)
- 類型: `number(正整數)`
- 預設: `711`

### radar-dispaly-hours

- 說明: 循環顯示的雷達回波小時。
- 類型: `number(正整數)`
- 預設: `1`

## 範例

`https://status.exptech.dev/weather-dashboard/dashboard?region=100&radar-dispaly-hours=3`
